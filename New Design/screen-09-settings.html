What changed and why

  • Plain sans-serif "Settings" title. Lora-italic was right for Bookmarks (पोथी), Festivals (पञ्चाङ्ग), and Library — heritage-charged screens. Settings is utility. Restraint here makes the heritage flourish elsewhere mean more.
  • Section labels in small caps with a hairline rule. "Reading / Notifications / Data / About / Reset" — consistent with section labels everywhere else in the app. Scannable in < 2 seconds.
  • Removed all per-row icons. Bell, trash can, language, mortarboard, info, book, shield, document, mail — your current settings has 9 icons doing decorative work the labels already do. The redesign keeps icons only on the theme segmented control (where they convey state visually) and the external-link arrow on Privacy/Terms (which signals "leaves the app").
  • Theme picker becomes a 3-segment toggle (Auto/Light/Dark). Replaces the colored pill row in your screenshot. Active segment fills with saffron. Single tap to change; no menu.
  • Font-size slider with 7 ticks. Discrete steps with visible tick marks. Bookend "A / A" sized to communicate the range.
  • Added: Sanskrit display preferences. Devanagari + IAST diacritics on/off. This is a real preference for the audience (some users find IAST helpful, some find it cluttered). Opens a sub-screen.
  • Added: Storage info. "72.1 MB · 1,33,613 verses cached offline" — both reassurance and a key authenticity signal for a fully-offline app. Indian-format number grouping.
  • Added: Export bookmarks. Saves the pothī as a JSON file. Cheap to build, real user trust.
  • Added: Festival alerts toggle. The day-before notification for major parvas. Most users will want this.
  • Reset section at the bottom, with iron-red label. Destructive action. Confirmation dialog uses iron-red destructive button with explicit body text about what's preserved (bookmarks, notes).
  • About footer — सनातन / Sanatan Guide / v1.0.0 · build 1. Demoted from a primary settings row to a centered footer block. The version is still accessible to anyone who needs it for support, but it stops competing with actionable rows.
  • Time picker as a bottom sheet. Standard Material/iOS pattern with a saffron AM toggle. Centered time ("07 : 00") in serif, peripheral hours visible at reduced opacity.
